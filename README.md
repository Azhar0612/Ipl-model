# 🏏 IPL Match Prediction Model

A Machine Learning project that predicts the outcome of IPL (Indian Premier League) matches based on historical match data. This project uses data analysis and predictive modeling to estimate winning probabilities.

---

## 📌 Overview

The IPL Match Prediction Model analyzes past match data such as teams, venue, runs, wickets, and overs to predict the probability of a team winning a match. It demonstrates the application of machine learning in sports analytics.

---

## ⚙️ Features

- 📊 Data preprocessing and feature engineering  
- 🤖 Machine Learning model for match prediction  
- 📈 Win probability prediction  
- 🧠 Trained model stored using `.pkl`  
- 🏏 Real-time prediction inputs (team, score, overs, etc.)

---

## 🚀 How to Run

1. Clone the repository:
git clone https://github.com/Azhar0612/ipl-match-predictor.git

2. Navigate to the project folder:
cd Ipl_model

3. Install dependencies:
pip install pandas numpy scikit-learn

4. Run the script:
python ipl_model.py

---

## 📊 How it Works

1. IPL dataset is loaded and cleaned  
2. Features like teams, score, overs are processed  
3. Model is trained using classification algorithm  
4. Model predicts winning probability of teams
